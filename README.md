### About This Project

This project is a **complete rewrite** of the original
👉 [https://github.com/vakkko/online-zoo](https://github.com/vakkko/online-zoo)

The initial version was built using **HTML, SCSS and vanilla TypeScript**.
In this version, the application has been rebuilt from scratch using **React**.

### Purpose

This rewrite was created as part of the **RS School challenge**, with the goal of:

- Improving code structure and scalability
- Refactoring the project into reusable components
- Applying better state management and architecture
